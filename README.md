# EB Suite — GitHub Web

Static website for [eb-suite.com](https://eb-suite.com) — Full-Stack Digital Studio in Hong Kong.

## Pages

- **index.html** — Home
- **services.html** — Services
- **projects.html** — Projects
- **about.html** — About Us
- **contact.html** — Contact

## Tech Stack

Pure HTML + CSS (no build step required). Hosted on **GitHub Pages**.

## Setup

No build required. Open any `.html` file directly in a browser, or serve with any static server:

```bash
npx serve .
# or
python -m http.server 8000
```

## Deployment

Pushed to the `main` branch. GitHub Pages serves the site automatically at:

**https://eb-suite.github.io/github-web/**
